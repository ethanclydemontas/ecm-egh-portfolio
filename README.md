# portfolio website

A website curated by Edrian G. Huelva & Ethan Clyde Montas.

## features

- optimized for accessibility on mobile, tablet, and desktop.
- uses css variables.
- javascript rendering of members.
- effects, animations, palettes.

## file structure

`index.html`: main entry point.

`style.css`: contains all styles, variables, animations.

`script.js`: handles content interactivity.

## setup

1. clone the repository.
2. open `index.html` in your web browser.
